1. Introduction

The Mobile Calculator Application is a robust and user-friendly Android app designed to handle basic arithmetic operations, including addition, subtraction, multiplication, and division. The app is built using the MVVM (Model-View-ViewModel) architectural pattern to ensure a clean, modular, and maintainable codebase. The primary goal of this project is to provide users with a reliable tool for quick calculations while demonstrating good software development practices.

2. Methodology

Why MVVM

The Model-View-ViewModel (MVVM) architecture is a modern design pattern that separates the business logic (Model) from the user interface (View) through a ViewModel, which acts as a mediator. This pattern is especially useful in Android development, where maintaining a separation of concerns can enhance testability, reusability, and maintainability..
